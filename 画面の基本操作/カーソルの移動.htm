一覧の下側は、【本文】の画面です。先に説明しました通り、場合によって図書の内容や、図書の凡例が表示されます。
※表示オプションの【画面の分割形式】を縦三分割画面に変更している場合は、
　一覧の右側が本文参照画面になります。

図書の内容一覧で Enterを押すと、図書の内容を参照できます。
参照の開始と同時に、図書の内容を読み上げ始めます。読み上げを途中で停止したいときは、Escapeを押してください。読み上げが中断され、カーソルは停止させた位置にあります。
読み上げが最後まで終わると、カーソルは本文画面の最後にあります。そこからカーソルを移動しながら、内容をじっくりと読み上げ、確認することもできます。
参照を終わって、一覧に戻りたいときは、Escapeを押してください。

図書一覧で Spaceを押すと、選択している図書の凡例を参照できます。
こちらの場合は、読み上げが終わると、自動的に一覧に戻ります。

Tabを押すと、本文の画面に移動しますが、読み上げは行われません。
次に説明します、カーソルの移動コマンドや読み上げ操作で、自由に読み上げさせることができます。

◆カーソル移動の基本は、← →で左右の文節へ、↑↓で上下の段落へ移動です。
　他にも、次のコマンドで、自由に移動することができます。
　・次の文字へ移動　　　　　　　Ctrl + →
　・前の文字へ移動　　　　　　　Ctrl + ←
　・前の行へ移動　　　　　　　　Ctrl + ↑
　・次の行へ移動　　　　　　　　Ctrl + ↓
　文章の先頭から、改行文字で行が改められるまでのブロックが段落です。
　・段落の先頭へ移動　　　　　　Home
　・段落の末尾へ移動　　　　　　End
　文章の先頭が文頭で、最後が文末です。
　・文頭へ移動　　　　　　　　　Ctrl + Home
　・文末へ移動　　　　　　　　　Ctrl + End

◆フリーカーソルモードでカーソルの移動単位が変更できます。
　・フリーカーソルモード オン オフ 切り替え　Shift + Insert

　オンの場合、カーソルは次のように移動します。
　・→　　次の文字へ移動
　・←　　前の文字へ移動
　・↑　　前の行へ移動
　・↓　　次の行へ移動
　・Home　行の先頭へ移動
　・End 　行の末尾へ移動
　このようにカーソルは １行 １文字単位の移動に変わるので、文章をじっくり確認したい場面でオンにします。
　また、拡大画面をお使いの場合、通常のカーソル移動では、文字が画面外にあり表示されないことがあります。
　この場合、モードのオンで、カーソルの移動に伴い、画面外の文字も画面内にスクロールするようになり、すべての文字が表示できます。

◆現在のカーソル位置を音声で知るには、F9を押します。
　行 列の順で『5の1』のように読み上げます。
　点字図書の場合は、巻数とページ番号を『5巻 1ページ』のように読み上げます。

◆カーソル移動時の文字の読み方を変更するには、F8を押します。
　カーソル移動したときの文字の読み方には、
　簡易読み　　　　　簡単な 易しい説明の読みです。
　音訓読み　　　　　六点漢字の 音読みと訓読み で表現した読みです。
　音読み　　　　　　音読みです。
　詳細音訓読み　　　簡易読みよりも詳しい読みです。
　詳細区分読み　　　詳細音訓読みに､漢字の区分を加えた読みです。
　識別読み　　　　　全角 半角や、文字の種類も加えた詳しい読みです。
　ビープ　　　　　　簡単な電子音を出力します。
　と、７つの読み方があります。
　F8で、これらの読み方を順番に切り替えることができます。

◆カーソル移動の点字コマンドは、次の通りです。
　・↑　　　　　　　　　　　2 5の点 + Space
　・↓　　　　　　　　　　　3 6の点 + Space
　・→　　　　　　　　　　　2の点 + Space
　・前の行へ移動　　　　　　2 5の点 + 無変換
　・次の行へ移動　　　　　　3 6の点 + 無変換
　・段落の先頭へ移動　　　　せ + 無変換（ 段落の先頭の せ ）
　・段落の末尾へ移動　　　　ら + 無変換（ 段落のラストの ら ）
　・文頭に移動　　　　　　　と + Space（ 文章のトップの と ）
　・文末に移動　　　　　　　ら + Space（ 文章のラストの ら ）
　・カーソル位置の読み上げ　か + Space（ カーソル位置の か ）
