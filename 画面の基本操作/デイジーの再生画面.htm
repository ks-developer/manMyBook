マイブックで取り扱えるデイジーは次の通りです。
　1.CDで図書が提供される デイジーCD
　2.サピエ図書館などネット配信されているデイジー図書
　3.マイブックの本棚のようにディスクに保存したデイジー図書
このように複数の媒体が取り扱えますが、どれも同じ操作方法でお楽しみいただけます。
ただ、注意していただきたいことは、
デイジーには、音声デイジーとマルチメディアデイジー、それにテキストデイジーの3種類の形式があります。
これらは、再生方式が次のように異なります。
【音声デイジーとマルチメディアデイジー】は、話者の録音データを再生します。
【テキストデイジー】は、テキストデータを音声で読み上げます。

再生方式により、コマンドとショートカットキーが異なりますので、２つに分けて説明します。

1.【音声デイジーとマルチメディアデイジー】
　再生中、一時停止中に、次のコマンドを使用できます。
　・再生の一時停止と再開　　　　　　Space
　・次のフレーズ　　　　　　　　　　→
　・前のフレーズ　　　　　　　　　　←
　・次の見出し　　　　　　　　　　　↓
　・前の見出し　　　　　　　　　　　↑
　・見出し移動のレベル切り替え　　　Ctrl+フルキーの数字キー　※以降、補足１を参照
　・次のページ　　　　　　　　　　　PageDown
　・前のページ　　　　　　　　　　　PageUp
　・ページジャンプ　　　　　　　　　Ctrl + J
　・次のしおり　　　　　　　　　　　M
　・前のしおり　　　　　　　　　　　Shift + M

　・ミュート　　　　　　　　　　　　テンキーの数字 0
　　※ミュートとは、音を消したまま再生させることです。
　・音量を上げる　　　　　　　　　　Ctrl + ↑
　・音量を下げる　　　　　　　　　　Ctrl + ↓
　　※この音量は、再生音の大きさであり、PC-Talkerの音量には影響しません。
　・再生スピードを上げる　　　　　　Ctrl + →
　・再生スピードを下げる　　　　　　Ctrl + ←
　・再生スピードを標準に戻す　　　　Ctrl + テンキーの数字 0
　　※録音データの形式により、変更できない場合もあります。
　・右スピーカーのバランスを上げる　Ctrl + Shift + →
　・左スピーカーのバランスを上げる　Ctrl + Shift + ←
　　※これは、ステレオスピーカーの音量バランスを調整する機能です
　・カーソル位置 読み上げ 　　　　　F9
　　※現在のページ番号と 残り再生時間を読み上げます。

　・再生の終了　　　　　　　　　　　EscapeまたはBackSpace

2.【テキストデイジー】
　読み上げ中、読み上げ停止中、共に次のコマンドを使用できます。
　・次のフレーズ　　　　　　　　　　→
　・前のフレーズ　　　　　　　　　　←
　・次の段落　　　　　　　　　　　　↓
　・前の段落　　　　　　　　　　　　↑
　・次のページ　　　　　　　　　　　PageDown
　・前のページ　　　　　　　　　　　PageUp
　・次の見出し　　　　　　　　　　　Ctrl + PageDown
　・前の見出し　　　　　　　　　　　Ctrl + PageUp
　・次のしおり　　　　　　　　　　　M
　・前のしおり　　　　　　　　　　　Shift + M
　・読み上げの終了　　　　　　　　　EscapeまたはBackSpace

　・音声スピード切り替え　　　　　　F7
　・音声ボリューム切り替え　　　　　F11
　
　・カーソル位置 読み上げ 　　　　　F9
　　※合計のページ数と 現在のページを読み上げます。

　読み上げ中は、次のコマンドを使用できます。
　・次の空白の行　　　　　　　　　　Space
　・段落の先頭　　　　　　　　　　　Ctrl + ←
　・次の記号段落（●や①など）　　　Ctrl + ↓
　・前の記号段落（●や①など）　　　Ctrl + ↑
　・次の見出し段落（1.や1-1.など）　Shift + ↓
　・前の見出し段落（1.や1-1.など）　Shift + ↑
　・次の記号段落／見出し段落　　　　Ctrl + Shift + ↓
　・前の記号段落／見出し段落　　　　Ctrl + Shift + ↑

　読み上げ停止中は次のコマンドを使用できます。
　・読み上げ再開　　　　　　　　　　Space


※補足１
音声デイジーとマルチメディアデイジーの再生中、上下矢印キーの見出し移動にて、移動する見出しレベルを切り替えることが可能です。
切り替えるには、デイジー再生中に Ctrl+フルキーの数字キーを押します。
次のようにご利用ください。
再生の開始時は、上下矢印キーを押すと、レベルに関係せず、すべての見出しに移動します。
これを切り替えるには、Ctrl+フルキーの数字キーを押します。
数字が見出しのレベル番号になりますので、例えば、Ctrl+3キーを押すと、レベル３の見出しが移動の上限に切り替わります。
上限とは、上位のレベルも含んだ番号という意味になりますので、上記の例では、レベル１、レベル２、それにレベル３の見出しに移動します。
移動をすべての見出しに戻したい場合は、 Ctrl+0を押します。または、図書を開き直しても戻ります。
なお、デイジー規格では、レベル１から６までの見出しを付けることができますので、Ctrl+1 から Ctrl+6 までのキーが割り付いています。
ただし、図書により、付けられているレベルの上限が異なりますので、図書によっては、Ctrl+6のような、付けられていない番号への切り替えはできません。
