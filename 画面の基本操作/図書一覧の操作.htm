一覧を進んでいくと、図書の一覧になります。図書一覧には 図書が並び、選択して図書の題名などを確認できます。図書の本文は、読み上げ機能を使って読むことができます。

図書一覧は簡単なキー入力で操作できます。

　・上の図書を選択　　　　　　　　　↑
　・下の図書を選択　　　　　　　　　↓
　・選択している図書の内容読み上げ　→ または Enter
　　　　　　　　　　　　　　　　　　→ で始めた場合は ← で停止
　　　　　　　　　　　　　　　　　　Enter で始めた場合は Escape で停止
　・選択している図書の凡例読み上げ　Space
　　　　　　　　　　　　　　　　　　Escape で停止
　・前の一覧へ戻る　　　　　　　　　← または Escape または BackSpace
　・戻った前の一覧から進む　　　　　Shift + Escape または Shift + BackSpace

上下左右の 矢印キーを覚えておくだけで、図書一覧の操作から図書を読むまでの操作ができますね。
この他、一覧の操作に共通して、次のキー操作ができます。

　・先頭の図書へ移動　　　　Home
　・最終の図書へ移動　　　　End
　・上へスクロール移動　　　PageUp
　・下へスクロール移動　　　PageDown

◆図書の種類によっては、一覧中に「ヘッダ」と呼ばれる項目があります。
【すべて】の一覧にある、フォルダとファイルの区切りなどです。選択している図書の種類によって、機能は異なります。
各ヘッダの位置へは、次のジャンプコマンドで素早く移動できます。
　・前のヘッダ　　　　　　　Ctrl + PageUp
　・次のヘッダ　　　　　　　Ctrl + PageDown

また、それぞれの図書は、既読・未読の状態を設定することができます。
選択している図書は 内容を読み上げたとき、自動的に、既読状態となりますが、次のコマンドで、既読・未読の状態を変更できます。
読んでいない図書を選択したときは、『未読』と音声ガイドが付きます。
　・既読・未読の切り替え　　Insert
　・既読状態にする　　　　　Ctrl + Q
　・未読状態に戻す　　　　　Ctrl + I
まだ、読んでいない図書を選択したいときは、次のコマンドでジャンプできます。
　・先頭の未読 図書　　　　 Ctrl + Home
　・最終の未読 図書　　　　 Ctrl + End
　・前の未読 図書　　　　　 Ctrl + Shift + ↑
　・次の未読 図書　　　　　 Ctrl + Shift + ↓

※点字キー入力でお使いの場合は、点字キーコマンド一覧《一覧の操作》も参考にしてください。

◆図書の種類によっては、一覧を並べ替えることができます。
並べ替えられる一覧には「並び替え」の項目がありますので、選択して Enterを押します。
並び順の一覧になりますので、更に選択すると、一覧が並べ替えられます。

◆一覧情報読み上げ
一覧で、Alt + F1 を押して、一覧情報読み上げを実行した場合、
現在一覧しているジャンルの位置を読み上げ『100件中 8件目を選択しています』と、現在の状態も一緒に読み上げます。

◆選択した図書を読みたいときは、→ または Enterを押します。
その図書は自動的に 既読状態になります。
※《テキストの読み上げ画面》を参照してください。
